Sentiment Analysis of Tamil Regional Languages using Machine Learning Algorithms

Project Overview

This project focuses on analyzing sentiment in Tamil regional language text using Machine Learning algorithms. It classifies user input into **Positive, Negative, or Neutral** categories.

The system is designed to handle Tamil dialects such as **Kongu Tamil** and **Madurai Tamil**, making it useful for real-world regional language sentiment analysis.

 Features

* Sentiment prediction (Positive / Negative / Neutral)
* Supports Tamil regional datasets
* Multiple Machine Learning models used
* Flask-based web application
* Simple and user-friendly interface
  
🛠️ Technologies Used

* Python
* Flask
* HTML, CSS
* Scikit-learn

 Machine Learning Models Used

* Logistic Regression
* Naive Bayes
* Random Forest

📊 Dataset Details

* Tamil sentiment dataset
* Tamil offensive dataset
* Custom dataset (Kongu & Madurai Tamil)
* Data collected from social media, YouTube, and reviews

⚙️ Project Structure

* backend/ : Contains Flask API and Machine Learning model
* frontend/ : Contains user interface files

▶️ How to Run the Project

1. Clone the repository
2. Navigate to backend folder
3. Install dependencies:
   pip install -r requirements.txt
4. Run the application:
   python app.py
5. Open browser and go to:
   http://127.0.0.1:5000

📈 Sample Output

Input: "இந்த படம் ரொம்ப நல்லா இருக்கு!"
Output: Positive
