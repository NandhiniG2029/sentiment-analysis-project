About Tamil Sentiment Analysis

Tamil Sentiment Analysis is a project designed to analyze the sentiment of Tamil text using advanced machine learning techniques. Our goal is to provide an accurate and efficient way to classify Tamil text into positive, negative, or neutral sentiments.

Features

  • Upload Tamil text files for analysis
  • Predict sentiment of user-input comments
  • Generate sentiment reports with graphs
  • User-friendly interface

Technologies Used

  • Python & Flask (Backend)
  • Machine Learning for Sentiment Prediction
  • HTML, CSS (Frontend)
